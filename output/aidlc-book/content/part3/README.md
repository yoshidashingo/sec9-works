# Part III: Inceptionフェーズ（計画編）

## 概要

本パートでは、AI-DLCの最初のフェーズである **Inceptionフェーズ** を、実際のECサイト開発プロジェクトを通して体験します。

INCEPTIONフェーズは、プロジェクトの「起点」です。ここで何を決めるかが、その後のCONSTRUCTIONフェーズの品質を大きく左右します。

## サンプルプロジェクト

本パートを通じて、以下のサンプルプロジェクトを使用します。

| 項目 | 内容 |
|------|------|
| **プロジェクト名** | BookCart |
| **概要** | 書籍・雑貨を販売するB2C物販サイト |
| **テックスタック** | TypeScript + Next.js（App Router） |
| **規模感** | 中小企業向け、商品数1,000点程度 |

> **Chapter 8のみ例外**: Brownfieldシナリオとして「LegacyShop」（Node.js + MySQLの既存ECサイト）を使用します。
