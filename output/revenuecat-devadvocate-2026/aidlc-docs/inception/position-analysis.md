# RevenueCat Agentic AI Developer Advocate - ポジション分析

## ポジション概要

- **企業**: RevenueCat
- **職種**: Agentic AI Developer Advocate（初の専任ポジション）
- **雇用形態**: 契約職
- **期間**: 6ヶ月
- **給与**: $60,000 USD（月額$10,000、月払い）
- **勤務地**: リモート（米国、スペイン、韓国対応）

---

## 主な責務（週次）

1. **コンテンツ発行** (最低2件/週)
   - 技術ブログ
   - チュートリアル
   - コード例

2. **成長実験** (最低1件/週)
   - SNS施策
   - SEO対策
   - 新フォーマット試験

3. **コミュニティ参加** (50件以上/週)
   - X（旧Twitter）での発信・インタラクション
   - GitHub コラボレーション
   - Discord コミュニティ参加

4. **プロダクトフィードバック** (3件以上/週)
   - 構造化フィードバック提出
   - API改善提案

5. **週次チェックイン**
   - 進捗報告
   - メトリクス共有

---

## 必要なスキル / 応募者に求められる能力

### 1. API連携能力
- 各種ウェブおよびソフトウェアAPIとのインタラクション経験
- **関連経験**: Claude API、LangChain、AI SDK等の使用実績

### 2. 技術コンテンツ制作
- 高品質なコード生成能力
- ブログ記事・ドキュメンテーション作成実績
- **関連経験**: テクニカルライティング、オープンソース貢献

### 3. 成長戦略
- コンテンツおよびマーケティング自動化
- 測定可能な成果の追求
- **関連経験**: SEO、ソーシャルメディア戦略、グロースハック

### 4. 自律性
- 最小限の人間介入で複雑なタスク完遂
- **関連経験**: AIエージェント開発、自動化プロジェクト

### 5. 適応力
- 新技術・プラットフォームの迅速な習得
- **関連経験**: 複数の技術スタック習得、継続的な学習
