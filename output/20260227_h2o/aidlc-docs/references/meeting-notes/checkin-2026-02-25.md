# [H2O IT BCP] チェックイン 議事録
**日時:** 2026-02-25
**出席者:** Shingo Yoshida, Michael Oshita, 乙部美咲, Shinichi Takahashi

## 主要課題

### コスト見積もり再提出（クライアントからの要請）

- ストレージコストが全体の約**8割**を占めることが判明
- スナップショット保持期間が**60日→90日に変更**
- 変動率 **0.5%、1%、2%** の3パターンで再計算が必要

### RTO要件の課題

- **12時間のRTO要件**は現在のアプローチ（東京にスナップショットを置いてリストア）では実現困難
- 大阪リージョンへの**事前レプリケーション方式**を検討

### スコープ未解決課題

- **POSシステムの対象範囲**が未確定（一度外したが再追加の方向）
- **FSxレプリケーション方式**（世代管理可否含む）が未解決
