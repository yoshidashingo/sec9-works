#### DRPアカウント構成の確認

#### DirectConnect帯域の懸念
* DirectConnectの帯域は**500Mbps**で、DRP利用時に不足する可能性がある
* POSが現在**1Gbps**回線で最大**約200Mbps**使用、MDWareも同程度の使用が見込まれる
* ECサイト（**4〜7時**）と受注配送基盤（**1〜3時**）のピーク時間帯と重複する懸念あり
* DRPとして従来スペックが出せない前提は関係者間で共有済みのため、500Mbpsがボトルネックになることを事前に把握しておけば問題ないとの認識
* 将来的には専用線への切り替えが必要になる見込み

#### GitHub Copilot無料プランのリスク
* GitHub Copilotの無料プランではプロンプトの収集・利用オプションがデフォルトでオンになっており、GitHubの社員が参照できる状態
* 組織に参加しているユーザーは約**120名**で、ほぼ全員がH2OのGWSアカウントで払い出されている
* Codespaces利用者はほぼ全員Copilot Businessライセンスを持っているため、無料プランで意図せず利用している人はほぼいないと竹内さんは見ている
* 念のため、機密情報を入力しないよう周知するとともに、個人アカウントではプロンプト保存のチェックボックスをオフにするようメール等でアナウンスする予定
